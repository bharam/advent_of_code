# Advent of Code 2023 (Rust)

## Instructions

- Run: `cargo run -p <day> --bin <part>` or `just run <day> <part>`
- Test: `cargo nextest -p <day> <part>` or `just test <day> <part>`
- Arguments: 
    - day: `day-xx` (i.e., `day-03`)
    - part: `part1 | part2`
